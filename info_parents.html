Uniformes

Code vestimentaire pour l'année scolaire 2025-2026

  • Haut : Polo blanc avec le logo de l'école
  • Bas : Pantalon bleu marine ou jupe bleu marine
  • Chaussures : Noires, fermées et confortables
  • Accessoires : Cravate optionnelle pour les occasions spéciales

Les uniformes peuvent être achetés à la boutique de l'école ou chez les fournisseurs locaux. Il est important que les élèves portent leur uniforme propre et bien entretenu chaque jour.

Les conséquences de ne pas respecter le code vestimentaire sont les suivantes :

  • Avertissement verbal pour la première infraction
  • Avertissement écrit pour la deuxième infraction
  • Exclusion temporaire pour les infractions répétées
